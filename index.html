      localStorage管理

      localStorageの読み込みに失敗した場合、localStorageのリロードを試してみてください。
      削除したlocalStorageは復元できないので注意してください。（Export Fileがあれば復元できます）
      Export local storage data to fileでLocal Storageの情報をファイル出力できます。
      Import local storage data from fileで出力したファイルからLocal Storage情報を復元できます。

      注意事項

      • 当サイトはJavascript及びlocalStorageを利用しています。Javascript及びlocalStorageまたはその一部が利用できない場合、正常に利用できない場合があります。
      • 当サイトはHTML5に依存した機能を利用しております。最新のブラウザではない場合、一部または全ての機能が利用できない可能性があります。
      • 当サイトはPC（非タッチデバイス）に依存した機能を利用しております。タッチデバイス環境下では一部または全ての機能が利用できない可能性があります。
      • 利用の結果生じた損害について、一切責任を負いません。
      • あなたの適用される法令に照らして、本サイトの利用が合法であることを保証しません。
      • コンテンツとして提供する全てのデータについて、内容の合法性・正確性等、あらゆる点において保証しません。
      • 事前の予告無く、コンテンツの提供を中止する可能性があります。

      既知の不具合

      • 一部隊に4ユニット以上設定しようとすると、部隊構成とデータ算出結果の整合性が取れない。
        不具合回避策：ユニット数が3より多く設定しないようにしてください。

      注意事項

      当ツールはPC（タッチ以外で操作できる端末）向けになります。スマートフォン、タブレット、一部のタッチデバイスでは正常に動作しません。

      localStorage情報全削除

      localStorage情報を全削除するとユニット一覧、各防衛デッキ情報が全て削除されます。
      一度削除するとlocalStorage情報の復元は行えません。localStorage削除後、新しくlocalStorage情報を作成します。

      Chromeの場合、
      「C:\Documents and Settings\ユーザ名\Local Settings\Application Data\Google\Chrome\User Data\Default\Local Storage」
      辺りに格納されます。

      Firefoxの場合、Cookieと同じ場所に格納されるっぽいです。IEとEdgeはよくわかんないです。
